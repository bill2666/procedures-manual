    Conventions¶

    Code blocks

    All code uses fenced blocks with a language tag (vba, m, dax, python). Every code block has a copy button — click it to grab the snippet.

    Gotchas are first-class citizens

    If something is fragile or has a known edge case, it's documented explicitly in a warning box like this one. Don't bury it in a code comment.
